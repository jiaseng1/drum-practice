# 会计凭证生成工具 - Gitee Pages 部署

## 部署步骤

### 1. 上传到 Gitee 仓库

将 `index.html` 文件上传到你已有的 Gitee 仓库根目录。

### 2. 启用 Gitee Pages

1. 进入仓库「服务」→「Gitee Pages」
2. 部署分支选择你的默认分支（如 master/main）
3. 部署目录填写 `/`（根目录）
4. 点击「启动」按钮

### 3. 访问网址

启动后 Gitee 会分配一个访问地址，如：
`https://你的用户名.gitee.io/仓库名`

将该地址分享给同事，浏览器打开即可使用。

## 使用说明

- 所有数据处理在浏览器本地完成，**不上传任何数据到服务器**
- 首次加载需要联网（加载 SheetJS 库 ~500KB），后续可离线使用
- 支持四种凭证类型：计提工资、发放工资、结转工资、代扣费用
- 上传 Excel → 自动识别 → 一键生成 → 下载凭证 Excel
